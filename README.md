# FileFinderPro — Умный поиск файлов с графическим интерфейсом

FileFinderPro — это современная утилита на Python с графическим интерфейсом (GUI), предназначенная для быстрого и удобного поиска файлов на компьютере. Программа позволяет фильтровать поиск по категориям (документы, медиа, архивы, системные ключи), сохраняет историю путей и поддерживает смену тем оформления.

## Что делает программа

- **Глубокое сканирование:** Выполняет поиск файлов в выбранной директории и всех вложенных подпапках, используя многопоточность для сохранения отзывчивости интерфейса.
- **Фильтрация по категориям:** Позволяет искать не просто по имени, а по группам расширений (Office, PDF, Изображения, Видео, ЭЦП и др.) благодаря встроенной базе форматов.
- **Умная история:** Запоминает папки, в которых производился поиск, и позволяет быстро переключаться между ними через диалоговое окно истории.
- **Управление файлами:** Позволяет открыть найденный файл или перейти к его расположению в проводнике двойным кликом или через контекстное меню.
- **Персонализация:** Поддерживает переключение между Светлой и Тёмной темой с адаптацией всех элементов интерфейса.

## Для кого этот проект

- **Для офисных сотрудников**, которым нужно быстро находить документы Word, Excel или PDF в запутанных архивах.
- **Для системных администраторов и бухгалтеров**, которым требуется поиск специфических файлов (логов, ключей ЭЦП, конфигов).
- **Для обычных пользователей**, желающих иметь более удобную и быструю альтернативу стандартному поиску Windows.
- **Для разработчиков**, изучающих создание современных настольных приложений на Python с использованием PyQt6.

## Технологии и библиотеки

- **Python 3.x**
- **PyQt6** — для построения отзывчивого графического интерфейса.
- **QThread** — для реализации асинхронного поиска без зависания окна.
- **JSON** — для хранения настроек, истории и базы расширений.
- **OS / Subprocess** — для взаимодействия с файловой системой и запуска файлов.


## Использование:
1. Выберите папку для поиска (кнопка "Обзор" или выбор из "Истории").
2. Выберите категорию файлов слева (например, "Word" или "Изображения") или оставьте "Все файлы".
3. Введите часть имени файла в строку поиска и нажмите Enter (или кнопку поиска).
4. Дважды кликните по найденному файлу, чтобы открыть его.

## Преимущества

- **Скорость и удобство:** Мгновенный отклик интерфейса и быстрая фильтрация результатов.
- **Гибкость:** Возможность тонкой настройки расширений через файл `extensions.json` без изменения кода.
- **Визуальный комфорт:** Продуманный дизайн с цветовым кодированием путей и поддержкой темной темы для работы в ночное время.
- **Автономность:** Программа не требует интернета и хранит все настройки локально.
